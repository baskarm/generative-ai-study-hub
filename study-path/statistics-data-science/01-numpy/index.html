    1. Introduction to NumPy

    NumPy (Numerical Python) is a core Python library for scientific computing, offering fast, flexible multi-dimensional ndarray objects and a suite of mathematical tools(numpy.org, w3schools.com, stackoverflow.com, geeksforgeeks.org).
    Arrays are stored in contiguous memory, enabling vectorized operations that run orders of magnitude faster than pure Python loops(geeksforgeeks.org).

    2. NumPy Functions

    NumPy provides builtâ€‘in functions for creating and summarizing data(programiz.com):

    • Creation tools:
    • np.arange(start, stop, step)
    • np.linspace(start, stop, num)
    • Reshaping & structure: np.reshape, np.flatten
    • Aggregate statistics: np.sum, np.mean, np.std, np.min, np.max

a = np.arange(1, 5).reshape(2,2)
print(np.sum(a), np.mean(a), np.std(a))

    3. Trigonometric & Arithmetic Operations

    NumPy supports fast universal functions (ufuncs) for elementâ€‘wise mathematical operations(wesmckinney.com, en.wikipedia.org, programiz.com, codingnomads.com):

    • Trigonometric functions (angles in radians):
      np.sin, np.cos, np.tan, np.arcsin, np.arccos, np.arctan, np.degrees, np.radians

    • Arithmetic ufuncs: np.add, np.subtract, np.multiply, np.divide, np.power, and synonyms using Python operators.

    Example: 

    angles = np.array([0, np.pi/2, np.pi])
print(np.sin(angles), np.cos(angles))
print(np.power(angles, 2))

    4. Matrix Operations

    NumPy facilitates robust linear algebra operations(programiz.com, geeksforgeeks.org, medium.com, stackoverflow.com):

    • Matrix multiplication: np.dot(A, B) or the @ operator
    • Transpose: A.T
    • Identity & inverse: np.eye(n), np.linalg.inv(A)
    • Solving systems: np.linalg.solve(A, b)
    A = np.array([[1,2],[3,4]])
B = np.array([[2,0],[1,3]])
print(A.dot(B))
print(np.linalg.inv(A))

    5. Random Number Generation

    NumPyâ€™s random submodule lets you generate pseudorandom data for simulations and sampling(en.wikipedia.org):

    • np.random.rand(*shape) for uniform [0,1)
    • np.random.randn(*shape) for standard normal
    • np.random.randint(low, high, size) for integer samples
    np.random.seed(0)
print(np.random.randint(1,10, size=(2,3)))

    6. Indexing & Accessing Arrays

    Mastering indexing and slicing is essential for data selection:

    • 1â€‘D arrays: a[i], negative indices a[-1]
    • 2â€‘D arrays: a[i, j], whole row: a[0, :], column: a[:,1]
    • Boolean indexing: a[a > threshold]
    a = np.array([[1,2],[3,4]])
print(a[0,1], a[:,1], a[a>2])

    7. Modifying Array Entries

    NumPy allows direct mutation via broadcasted assignment or indexing:

    • Single-element assignment: a[0,1] = new_value
    • Row/column assignment: a[:,0] = [x, y, z]
    • Broadcasting across axis: fills patterns efficiently
    a = np.ones((3,3))
a[:,1] = np.arange(3)
a[0,2] = 99
print(a)

    8. Saving & Loading Arrays

    NumPy offers simple file I/O for persisting arrays(stackoverflow.com):

    • Save single: np.save('file.npy', array)
    • Load: np.load('file.npy')
    • Save multiple arrays to one file: np.savez('file.npz', a=a, b=b)
    np.save('data.npy', a)
a_loaded = np.load('data.npy')
